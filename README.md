# Linkedin Clone Project

Bu proje, Linkedin'in kullanıcı arayüzünün klonunu oluşturmayı amaçlar.

## Kullanılan Teknolojiler

Proje, aşağıdaki teknolojiler kullanılarak geliştirilmiştir:

- HTML
- CSS
- JavaScript
